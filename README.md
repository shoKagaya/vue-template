# popinfo-segment-form

> A Vue.js project

## 環境構築手順

# install dependencies
$ npm install

# serve with hot reload at localhost:8080
$ npm run dev

#### レイアウトクラス
`.l-`

例

`.l-header`

#### 再利用可能なモジュールクラス
`.mod-`

例

`.mod-btn`
`.mod-title1`
